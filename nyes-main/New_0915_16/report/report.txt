가위바위보 게임 만들기
1. 기본자금 1000원
2. 사람이 선택하기 전까지 컴퓨터의 선택 창이 계속 바뀜
3. 오른쪽 아래에 스타트 버튼, 클릭 시 게임 시작, 게임 시작 시 100원 차감, 컴퓨터의 표시 빠르게 변화
// 팀장급 : 동전 들어가는 모션
4. 아래에 가위 / 바위 / 보 버튼. 게임 시작 후 선택 가능, 선택 시 선택한 버튼 표시
5. 플레이어가 선택 시 천천히 멈춤, 컴퓨터가 뭘 선택했는지 가운데에 표시.
6. 이겼냐 졌냐 비겼냐 표시.
7. 이겼을 시 코인 갯수가 빙글빙글 빠르게 회전, 천천히 멈춤, 걸리는 만큼 코인 추가
