회원가입, 로그인, 로그아웃 추가
- 암호화 필요 없음
- 회원에 따른 게시판 목록 수정 필요 없음

(로그인했으면 회원가입, 로그인버튼 없어지고 로그아웃버튼 나오는것)

고수
- 데이터 저장(서버가 종료돼도 남아있어야함)

10.24
1. jwt를 사용한 로그인 확인
   - 웹 페이지 상에서 jwt의 데이터를 사용하여 사용자 출력

2. 로그인 시와 비로그인 시 웹 페이지 출력 다르게
 - 게시글에 대해 사용자 정보(이름, 닉네임 등등) 출력
 - 로그인 후
   - 글을 적을 수 있어야 한다.
   - 로그아웃 버튼 출력
   - 게시자만 수정 삭제 가능
 - 로그아웃 시
   - 로그인 가능
   - 게시글 작성 불가(UI가 없음)
   - 수정 삭제 불가

3. 일정 시간 후 자동 로그아웃

4. 서버 상에서 로그인 체크(정상적인 데이터를 확인해야함)

고수
- 미들웨어 활용

- cookie(jwt를 발행하여 쿠키에 저장), 미들웨어(둘째칠것, 힌트 : next)
